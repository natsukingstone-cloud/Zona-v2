# Zo-na Website - Version 2 (修正版)

蔵王町の温泉熱栽培いちごスイーツを浅草で提供する「Zo-na（ゾーナ）」の新しいウェブサイトデザインです。

## 🍓 講師フィードバック対応済み

### 修正内容

#### デザイン修正
✅ **FV背景画像をくっきり表示**
   - オーバーレイの透明度を下げ、背景画像がはっきり見えるように調整
   - キャッチコピーを白色+テキストシャドウで視認性を確保

✅ **FVのボタンを削除**
   - About、Menuのボタンを省いてデザインをすっきりさせました

✅ **画像の角を統一**
   - すべての画像を角丸（20px）に統一

✅ **フッターをシンプルに**
   - ネイビー背景のフッターを削除
   - © Zo-na を地図の下に余白を取って配置（白背景）

✅ **SNSアイコンをヘッダーに移動**
   - PC: メニューの右側に配置
   - SP: ハンバーガーメニューの中に配置

#### PC版修正
✅ **ヘッダーのメニュー色を赤に変更**
   - ナビゲーションリンクの色を黒から赤（--strawberry-red）に変更

✅ **About画像のレイアウト改善**
   - 画像が全体的に見えるよう、グリッドレイアウトを調整

#### SP版修正
✅ **ハンバーガーメニューのX形状を修正**
   - transform の数値を調整して正しいX形状に

✅ **SNSアイコンの配置**
   - SPではハンバーガーメニュー内の上部に配置

## 🎨 新デザインの特徴

- **クリアな背景画像**: FVの写真がしっかり見える
- **白ベースの洗練されたデザイン**: 余白を活かした高級感
- **セクションごとの配色**: 淡いピンクグラデーションで変化

## 🚀 セットアップ方法

### 1. ファイルの配置
1. ダウンロードした3ファイルを `Zona-sp-v2` に配置
2. 既存の `images` フォルダをコピー
3. `menu.html`、`favicon.ico`、`apple-touch-icon.png` もコピー

### 2. 確認ポイント
- [ ] FVの背景画像がはっきり見えるか
- [ ] FVのボタンが削除されているか
- [ ] すべての画像が角丸になっているか
- [ ] ヘッダーのメニューが赤色か
- [ ] SNSアイコンがヘッダーにあるか（PC）
- [ ] ハンバーガーメニューが正しく動作するか（SP）

## 📱 レスポンシブ対応

- **デスクトップ**: ナビゲーションメニュー + SNSアイコン表示
- **タブレット**: レイアウトが1カラムに
- **スマートフォン**: ハンバーガーメニュー + メニュー内にSNS
